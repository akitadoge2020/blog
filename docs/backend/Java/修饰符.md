# 修饰符

## 权限修饰符

在 Java 中，可以通过权限修饰符来控制对类中的属性和方法的访问。

权限修饰符一共有四种：

- public，公有的
- protected，受保护的
- 默认不写
- private，私有的



## 特征修饰符



- final，最终的
  - 属性一旦被赋值就不能被重新赋值
  - 方法不能被子类重写
- static，静态的
  - 属性和方法都独立于对象，通过类本身来调用
- abstract，抽象的
- native，本地的
- transient，瞬时的，短暂的
- synchronized，同步的，主要用于线程
- volatile，不稳定的，主要用于线程
