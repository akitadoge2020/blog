# 接口

用于约束类、对象、函数的契约(标准)。

契约(标准)的形式：

- api 文档，弱标准
- 代码约束，强标准

## 定义一个接口

格式：`interface 接口名称 { 接口成员: 类型 }`

```js
interface User {
  name: string
  age: number
  sayHello: () => void // 该字段的函数没有参数也没有返回值
  sayHello(): void // 另一种写法
  filter(item:number) => boolean // 该字段的函数参数需要是number类型，且返回布尔值
}
```

### 接口与类型别名的区别

- 接口与类型别名很相似，最大区别在于接口可以约束类
- 接口和类型别名一样，不会出现在编译结果中
